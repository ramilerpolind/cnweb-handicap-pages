# cnweb-handicap-pages

本仓库用于归档和发布一批独立 HTML 页面，这些页面不归属于任何特定域名或网站，而是作为独立的前端资源进行存储与分发。

## 项目简介

cnweb-handicap-pages 是一个静态页面集合仓库。其中收录的 HTML 文件均为独立页面，不依赖外部数据库或后端服务。每个页面均可通过直接打开或部署到任意静态托管服务来访问。

本仓库主要面向需要长期保留、可离线查看或用于技术参考的 HTML 页面归档场景。

## 目录结构

```
/
├── README.md          # 本文件
├── index.html         # 主入口页面（如有）
├── pages/             # 归档页面存放目录
│   ├── page-a.html
│   ├── page-b.html
│   └── ...
└── assets/            # 公共资源（样式、脚本、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：存放所有独立 HTML 页面文件。
- `assets/`：存放页面可能引用的公共静态资源。

## 页面归档说明

- 每个 HTML 文件均为完整的独立页面，包含必要的样式与脚本（内联或相对引用）。
- 页面之间无强制跳转或依赖关系，可单独使用。
- 归档文件保持原始结构与内容，不进行额外修改或优化，除非为修复明显错误。
- 所有页面均不使用第三方跟踪、统计或广告代码。

## 使用方式

1. 直接克隆或下载本仓库到本地。
2. 在浏览器中打开 `pages/` 目录下的任意 HTML 文件即可查看。
3. 也可将本仓库部署到任何支持静态文件的 Web 服务器或托管平台（如 GitHub Pages、Netlify 等）。
